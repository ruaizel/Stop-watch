Stopwatch
Chronos is a premium, web-based stopwatch application designed with a focus on minimalist aesthetics and high precision. 
Built using Flask and modern CSS techniques, it features a stunning glassmorphism UI and smooth animations.

✨ Features
  Glassmorphism UI: A beautiful, frosted-glass interface with backdrop blur and subtle gradients
  Precision Timing: Accurate tracking of hours, minutes, and seconds
  Interactive Controls: Sleek Lucide icons for Start, Pause, and Reset functionality
  State Animations: Dynamic glowing effects and transitions when the timer is active
  Responsive Design: Fully optimized for both desktop and mobile devices
  Premium Typography: Uses the 'Outfit' font for UI and 'JetBrains Mono' for the timer display
  
🛠️ Tech Stack
  * Backend: Python (Flask)
  * Frontend: HTML5, Vanilla CSS3 (Custom Design System)
  * Icons: Lucide Icons
  * Fonts: Google Fonts (Outfit, JetBrains Mono)
  * 
🚀 Getting Started
  Prerequisites
  Python 3.x
  Pip (Python package manager)
Installation
Install dependencies:
  - pip install flask

Run the application:

python app.py
  Open in your browser:
  Navigate to http://127.0.0.1:5000
